### Set Up Workflow
The item creation workflows is using standard Dynamics 365 functionality that you can use to ensure that approval process is approved in a consistent and efficient manner.
You can reach the Item creation workflows form by navigating to

**Item creation > Setup > Item creation workflows**
1 workflow must be created.

1.	Item creation import workflow - workflow must be created to create/update items in bulk via import. 

*Note: For information on how to create workflows please refer to the standard Dynamics 365 functionality.*
