#### 3.1.3	Setting up Item Creation Templates
#### 3.1.3.1	 Create Import File Masks

<b> Item creation > Setup > Item creation > Item creation import file masks </b>
Import file name masks need to be created in the system. You need to specify ma
sks for all the files you will be creating and using for the bulk item upload templates. 

<I> Note: Import file mask ‘Attrib*’ will look in the folder location specified for any files beginning with ‘Attrib’ </I>

1.	Click the <b> New </b> button. 
2.	In the <b> File Mask field </b> enter the name of the .csv file to be uploaded. The file mask can end with a * to allow for numbering of files. 
3.	The <b> Description </b> allows for a description of the file mask.
